# Particle Universe

> *"Du hast Zeit. Das Universum hat sie auch. Aber wie viel?"*

Ein minimalistisches Idle/Incremental Spiel, in dem du ein Universum erschaffst - von einem einzelnen Partikel bis zu allem.

## Konzept

Du beginnst mit einem einzelnen Partikel. Durch geschicktes Erschaffen und Vereinigen baust du es auf: Atom → Molekül → Stern → Planet → Leben → Zivilisation → Universum.

Bei jedem Schwellenwert ändern sich die ASCII-Visualisierungen und die narrativen Texte werden poetischer, existenzieller.

## Spielanleitung

### Buttons
- **Erschaffen** (C) - Erschaffe neue Partikel (kostet Partikel)
- **Vereinigen** (M) - Erhalte mehr Partikel pro Aktion (kostet mehr)
- **Transzendieren** (A) - Steige auf zur nächsten Stufe (hohe Kosten)

### Upgrades
Schalte upgrades frei, die deine Partikel-Produktion verbessern. Jede Stufe hat eigene Upgrades.

### Tastaturkürzel
- `C` - Erschaffen
- `M` - Vereinigen  
- `A` - Transzendieren
- `R` - Spielstand zurücksetzen

## Technisch

- Reines HTML/CSS/JavaScript
- Kein Framework nötig
- Läuft komplett im Browser
- LocalStorage für Speicherstände

## Entwicklung

```bash
# Projekt klonen
git clone https://github.com/oliverlaudan-ops/particle-universe.git
```
